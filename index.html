Değişken tanımı

Kaydedilmesi istenen tüm veriler(anlık hesaplamalar hariç) doğru bir isime sahip olmak zorundadır.

Değişken isimlerinde türkçe karakter, sembol, boşluk kullanılamaz ve sayı ile başlayamaz. 
1degişken = "......."
Doğru kullanım 
birinci_degisken = "......."

Değişken içeriği

Değişken içeriğine metin girilmesi gerekiyorsa tırnak işaretleri( "..." ) içerisinde olmak zorundadır. Aksi halde değişken içeriği bir metin olarak değil başka bir içeriğin adı olarak yorumlanır. 

birinci_degisken = "kerem"
Sonuç
kerem 
birinci_degisken = kerem
Sonuç
Metin olarak değil bir değişken adı olarak yorumladı ve kerem diye bir değişken olmadığı için hata verdi.

Aritmetik Operatörler

Kodlamada sonuç olarak en fazla 2 farklı sonuç yorumlanır. true(doğru) yada false(yanlış). İki sonuçta uymuyorsa sonuç nil(boş) olarak yorumlanır.
Genel operatörler
+ toplama
- çıkarma
* çarpma
/ bölme
** kuvvet
% mod alma
Sorgu operatörleri
> büyük ise
>= eşit veya büyük ise
< küçük ise
<= eşit veya küçük ise
== eşit ise
!= eşit değil ise
Örnek sorgu
Sonuç
if ( a=3 > b=5 )
 false   
if ( a=3 >= b=5 )
 false   
if ( a=3 < b=5 )
 true   
if ( a=3 <= b=5 )
 true   
if ( a=3 == b=5 )
 false   
if ( a=3 != b=5 )
 true  

if(ise) Sorgusu

if içerisinde bir bölümde aynı anda en fazla 2 şeyin karşılaştırması yapılabilir. Daha fazlası için &&(ve), ||(veya) operatörleri kullanılmalıdır. Sorguların bitişini belirtmek için sorgu sonucuna end girilmesi zorunludur.

Yanlış kullanım 
if (birinci_degisken > ikinci_degisken > ucuncu_degisken)
...
end
Doğru kullanım 
if (birinci_degisken > ikinci_degisken)   &&   (ikinci_degisken > ucuncu_degisken)
...
end
if (birinci_degisken > ikinci_degisken)   ||   (ikinci_degisken > ucuncu_degisken)
...
end

Else(değil ise) ile iki sonuçlu sorgulama yapma

Birinci koşul sağlanamıyorsa else ile değilse sorgu sonucu yazdırılabilir. 

a= 3, b= 5

if (a >= b )
    puts true
else
    puts false
end
Sonuç: 
false

Elsif(yada) ile İkiden fazla sonuçlu sorgulama yapma

Sorgu sonucu 2 farklı sonuçtan daha fazla ise birinci sorgudan sonra elsif ile devam edilir.

elsif sorgusu daima else sorgusundan önce yazılabilir. 

a= 3, b= 5

if (a >= b )
    puts true
elsif (a == b)
    puts "eşit"
else
    puts false
end
Sonuç: 
false

Sorgu Operatörleri

&& (ve) Operatörü
Aynı anda aynı bölümde &&(ve) operatörüyle birden fazla sorgu yapılıyorsa ve bu sorgulardan en az biri false ise sonuç daima false olacaktır. 
a= 3, b= 5

if (a <= b && a >= b )
  puts true
else
  puts false
end
Sonuç: 
false
a <= b true(doğru) olmasına rağmen ikinci bölümde a >= b false(yanlış) olduğu için çıktı sonucu yinede false olur.

|| (veya) Operatörü
Aynı anda aynı bölümde ||(veya) operatörüyle birden fazla sorgu yapılıyorsa ve bu sorgulardan en az biri true ise diğer sorgular ne olursa olsun en az 1 doğru olduğu için sonuç daima true olacaktır. 
a= 3, b= 5

if (a <= b || a >= b )
    puts true
else
    puts false
end
Sonuç: 
true
İkinci bölümde a >= b false(yanlış) olmasına rağmen birinci bölümde a <= b true(doğru) olduğu için çıktı sonucu true olur.
